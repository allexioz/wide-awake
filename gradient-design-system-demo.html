‚ôø Accessibility Considerations

The gradient animations respect the prefers-reduced-motion media query from your critical.css file. Users who prefer reduced motion will see static gradients instead of animations.

/* This is automatically handled by your existing CSS reset */ @media (prefers-reduced-motion: reduce) { .gradient-rainbow { animation: none !important; } }
